# Estimations

Use this skill to create practical engineering estimates with visible assumptions and uncertainty.

## Workflow

1. Clarify the desired outcome and definition of done.
2. Break the work into independently verifiable tasks.
3. Identify unknowns, dependencies, integration points, and review or release overhead.
4. Estimate using ranges when uncertainty is meaningful.
5. Separate must-have scope from optional or speculative scope.

## Guidance

- State assumptions that materially affect the estimate.
- Call out discovery work separately from implementation work.
- Include testing, review, deployment, and coordination time when relevant.
- Prefer honest ranges over false precision.
- Describe what would shrink or expand the estimate.

## Output

For estimation tasks, provide:

- Scope summary.
- Task breakdown.
- Estimate range.
- Key assumptions.
- Major risks and ways to reduce uncertainty.
